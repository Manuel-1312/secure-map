# Contribuye a Secure Map Codec

1. Abre un issue describiendo la mejora, la amenaza o el caso de uso.
2. Haz un fork, crea una rama descriptiva (`face2-tests`, `feature/package`, etc.).
3. Instala deps: `pip install -r requirements-dev.txt`.
4. Corre los tests: `pytest tests` y verifica la calidad con `ruff check core ui tests`.
5. Abre un PR y menciona qué Face toca. Revísalo usando el workflow `face2-quality.yml`.
